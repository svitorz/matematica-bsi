10. Conectivo SE... Então

'Se p então q' só sera falsa quando p for verdadeira e q for falsa.

| p   | q   | p->q |
| --- | --- | ---- |
| v   | v   | v    |
| -   | -   | -    |
| v   | f   | f    |
| -   | -   | -    |
| f   | v   | v    |
| -   | -   | -    |
| f   | f   | v    |
| -   | -   | -    |
